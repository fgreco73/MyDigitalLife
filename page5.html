GIT

Git : is a distributed revision control system. It is a computer program that helps people create other computer programs together. Git was made to be fast. It was created by Linus Torvalds for use in developing the Linux kernel which he also created. Git's current development is looked after by Junio Hamano. It is free and open source software released under the GNU General Public License version 2 software license. (Source Wikipedia)

Add all changes, commit, and push with a single command


This is very powerful and should be used with caution, especially on shared branches or for complex changes. It's ideal for quick fixes or custom features.

git add . && git commit -m "Description of commit" && git push

Add, commit, and push with a custom message (with alias)


You can create an alias to make it even more convenient and ask for the message
1) Add the alias below to your .gitconfig file
    [alias]
    myPersonalAlias = "!f() { git add . && git commit -m \"$@\" && git push; }; f"

2) Then you can use it like this
    git myPersonalAlias "Description "

Delete local branches already merged into the current branch (usually main or master)


This command removes local branches that have already been merged into the current branch, helping keep your local branch list clean and organized.

git branch --merged | grep -v "\*" | xargs -n 1 git branch -d
