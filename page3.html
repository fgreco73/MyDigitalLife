AWK - SED

awk: It is a programming language and a command-line tool, very powerful for processing an analyzing structured text, such as CSV files or logs.
sed: It is a command-line tool used to process and transform text streams. It’s often used for automated editing of files or outputs without opening them in a text editor.


Replace strings in a CSV file

template.csv : CSV file containing the substitution table.
file.csv : CSV file to be modified.

Ready to use (Linux version)
awk -F',' '{print "s|\\<" $1 "\\>|" $2 "|g"}' template.csv | sed -i -f - file.csv
