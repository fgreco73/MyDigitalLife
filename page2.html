Crunchy dispenser

This device is an automatic pet feeder designed for both dogs and cats. It dispenses dry food (kibble) at scheduled times, ensuring your pets are fed regularly even when you are not at home.

Through a simple control panel, users can set the exact times for two daily meals. 

The feeder is powered by a power bank, making it completely portable and independent from the electrical grid. Thanks to this feature, it can be used anywhere without needing a wall outlet.

Circuitry

The crunchy dispenser is made up of:
* an Arduino Nano
* a 16×2 LCD display using I²C
* 10 microswitch buttons and associated resistors.
* RTC board
* Stepper motor
* Stepper driver board
* Power supply module

Frontal panel control

The front panel allows you to manage the functions of the kibble dispenser. Using the yellow buttons, you can select the meal time and then verify the selected time with the info button. The green buttons allow you to set the time.

Display information

The display always shows the date and time.
Using the arrow buttons, you can choose the time for the first meal dispensing.
With the same arrow buttons, you can choose the time for the second meal dispensing.
Displays the times of the two selected meals to be dispensed.
