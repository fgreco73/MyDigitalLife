Shortcut keyboard

Device with nine programmable keys and a 16×2 LCD display that shows the executed command or the current status of the device. It can be used with either the macOS or Windows operating system. When powered on, the device displays on the screen the option to select the appropriate operating system for proper operation between the two systems mentioned above.

Circuitry

The shortcut keyboard is made up of:
* an Arduino Leonardo board
* a 16×2 LCD display using I²C
* 9 microswitch buttons.

Functionality on macOS

The set of keys on the right displays the configuration of the programmable keys for macOS. Pressing a key will trigger the assigned function or command.
